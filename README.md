# Chat for fun react

## Available Scripts

In the project directory, you can run:

`npm start`

`npm test`

`npm run build`

`npm run eject`

## Docker

### Run dev
`docker-compose up --build`

### Stop dev
`docker-compose down -v`

### Run prod
`docker-compose -f docker-compose.prod.yml up --build`

### Stop prod
`docker-compose -f docker-compose.prod.yml down -v`
